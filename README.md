# 名片小專案（高三多元選修：網頁設計與實作）

這是在高三多元選修課 **「網頁設計與實作」** 製作的名片網頁小專案。

## 線上預覽

- GitHub Pages：<https://lzspriv.github.io/namecard/>

## 作品內容

本專案包含兩個版本：

### 1) 主版本（根目錄）
位置：`index.html`、`style.css`、`script.js`

特色：
- 深色主題名片卡與背景遮罩
- 以 JavaScript 動態產生「專長興趣」與「技能條」
- 卡片懸停時有陰影與技能條漸層動畫

### 2) 名片練習無JS版本（`名片練習/`）
位置：`名片練習/index.html`、`名片練習/style.css`

特色：
- 名片正反面設計
- 滑鼠懸停可翻轉卡片（3D flip）
- 背面提供聯絡資訊（Email、網站、Instagram）
- 打字機文字效果與背景圖覆蓋濾鏡
